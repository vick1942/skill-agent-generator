# skill-agent-generator
Create skills and agents

Create a new repository on the command line
echo "# skill-agent-generator" >> README.md
```
git init 
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/vick1942/skill-agent-generator.git
git push -u origin main

or push an existing repository from the command line

git remote add origin https://github.com/vick1942/skill-agent-generator.git
git branch -M main
git push -u origin main
```
